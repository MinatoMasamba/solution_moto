# solution_moto
Solution Moto — Plateforme numérique de gestion de taxi-motos à Kinshasa. Elle relie clients, motards et propriétaires de flottes : mise en relation, suivi GPS des courses, gestion des motos, abonnements motards et collecte des paiements. Un écosystème adapté à la mobilité urbaine congolaise.

## Architecture Front-end
Les tableaux de bord utilisent des composants front-end dynamiques (ex: Sidebar, KPIs) qui récupèrent les données en temps réel via l'API, évitant ainsi le recours à des données statiques codées en dur.
